# EduGraph-AI

AI-Powered Adaptive Learning Assistant

EduGraph combines:

- Machine Learning (knowledge tracing, difficulty modeling)
- Knowledge Graphs
- RAG (Retrieval-Augmented Generation)
- Prompt Engineering

to provide **structured, personalized learning**.

---

## Features

- Concept Knowledge Graph (Variables → Loops → Functions → Recursion → DP)
- RAG-based contextual answering
- ML-based student modeling (prototype)
- Adaptive quiz difficulty prediction
- FastAPI backend

---

## Running the API

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
